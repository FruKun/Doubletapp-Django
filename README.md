# Template for backend course

### install dependencies

```sh
poetry install --no-root
```

### create .env

### create super user

```sh
make createsuperuser
```

### start web

```sh
make dev
```

### start bot

```sh
make bot
```

### start docker

```sh
make docker-up
```

### down docker

```sh
make docker-down
```

### logs docker

```sh
make docker-logs
```
